# Eingesetzter Sensor: Pedometer

## Allgemeine Eigenschaften des Sensors
Der Pedometer verwendet die Bewegungssensoren des Telefons, um zu erkennen, wann die Person sich mit dem Handy bewegt. Dabei wird versucht die Anzal der Schritte zu messen. Wenn der Schrittzähler auf die Länge des Fußes abgestimmt ist, können die Werte besser geschätzt werden.
